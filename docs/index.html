notion-team-dashboard

Auto-generate a weekly activity dashboard from your team's Notion workspace. Zero backend. Zero database. Just one script.

See Live Demo View on GitHub
$ ./generate_dashboard.sh
  → claude --print (Claude Code CLI)
    → Notion MCP (search, fetch per member)
    → dashboard.html ← single self-contained file

Features

📊

Stats Overview

Active projects, team members, upcoming deadlines, and pages updated at a glance.

👥

Member Activity Cards

Expandable cards for each team member showing their updated pages, to-do items, and direct Notion links.

📅

Deadline Tracker

Countdown badges for upcoming deadlines extracted from your Notion pages.

📋

Project Progress

Task breakdown per project: done, in progress, and to-do items with visual indicators.

⏰

Activity Timeline

Chronological view of all workspace changes across team members.

🔄

Fully Automated

Schedule with cron or launchd. Runs every Friday (or whenever you want) with zero manual effort.

How It Works

📝

1. Configure

List your team members and their Notion page IDs in config.json

🤖

2. Claude Queries Notion

Claude Code CLI uses the Notion MCP server to search and fetch each member's recent activity

🌐

3. Dashboard Generated

A single self-contained HTML file with dark theme, interactive cards, and direct Notion links

Quick Start

1

Create a Notion Integration

Go to notion.so/my-integrations, create an internal integration, and copy the token (ntn_...). Share your workspace pages with it.

2

Configure Claude Code with Notion MCP

Add to ~/.claude/settings.json:

{
  "mcpServers": {
    "notion": {
      "command": "npx",
      "args": ["-y", "@notionhq/notion-mcp-server"],
      "env": {
        "OPENAPI_MCP_HEADERS": "{\"Authorization\": \"Bearer ntn_YOUR_TOKEN\"...}"
      }
    }
  }
}
3

Clone & Configure

git clone https://github.com/joonan30/notion-team-dashboard.git
cd notion-team-dashboard
cp config.example.json config.json
# Edit config.json with your team members and Notion page IDs
4

Generate

./generate_dashboard.sh
open dashboard.html

Requirements

Claude Code CLI

The claude command must be installed and on your PATH. Get it here.

Node.js 18+

Required for the Notion MCP server (npx @notionhq/notion-mcp-server).

Notion Workspace

With an internal integration that has read access to team pages.

Python 3

For config parsing in the shell script. Pre-installed on macOS and most Linux distributions.
